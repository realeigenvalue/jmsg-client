# jmsg-client

A text based Java message client using Java Networking.

![Application Image](jmsg-client.png)

## Building

Prerequisites
- Java JDK
- Gradle Build Tool

```bash
git clone https://github.com/realeigenvalue/jmsg-client.git
cd jmsg-client
gradle build
gradle run
```
